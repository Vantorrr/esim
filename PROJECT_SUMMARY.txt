🚀 КАК ЗАПУСТИТЬ
═══════════════════════════════════════════════════════════════════

1. Установите зависимости:
   npm install

2. Настройте .env файл:
   - ESIM_GO_API_KEY (получите на esim-go.com)
   - STRIPE_SECRET_KEY (получите на stripe.com)
   - TELEGRAM_BOT_TOKEN (создайте через @BotFather)

3. Запустите:
   ./start.sh

   Или раздельно:
   npm run server:dev  # Backend на :8080
   npm run dev         # Frontend на :3000

4. Откройте: http://localhost:3000

Подробная инструкция: SETUP.md

🔑 НЕОБХОДИМЫЕ API КЛЮЧИ
═══════════════════════════════════════════════════════════════════

1. esim-go.com (обязательно)
   → Регистрация: https://esim-go.com
   → API docs: https://docs.esim-go.com
   → Получить ключ: Dashboard → API → Create Key

2. Stripe (для международных платежей)
   → Регистрация: https://stripe.com
   → Получить ключи: Developers → API keys
   → Настроить webhook: Developers → Webhooks

3. Т-Банк (рекомендуется для РФ) 💛
   → Подключение: https://www.tbank.ru/business/ecom/
   → Получить ключи: Личный кабинет → Терминалы
   → Инструкция: TINKOFF_SETUP.md

4. YooKassa (опционально, для РФ)
   → Регистрация: https://yookassa.ru
   → Получить ключи: Настройки → API

5. Telegram Bot (для Mini App)
   → Создать: @BotFather → /newbot
   → Настроить: /newapp → /setappurl
